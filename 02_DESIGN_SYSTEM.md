# Design System

## Prinsip

Modern, bersih, premium, mobile-first.

## Warna

Primary: #0F766E
Secondary: #14B8A6
Accent: #F59E0B
Background: #F8FAFC
Text: #111827

## Typography

Font: Inter
Heading: Bold
Body: Regular

## Radius

Card: 20px
Button: 14px

## Spacing

Gunakan grid 8px.

## Komponen

- Navbar sticky
- Hero fullscreen
- Glass card
- Hover lembut
- Skeleton loading
- Empty state
- Error state
- Success state

## Animasi

Gunakan Framer Motion.
Durasi 150–300ms.
Tidak berlebihan.
